## 🎯 Problem Statement

Bank branch staff in India struggle to communicate with customers who speak regional languages like Marathi, Hindi, Tamil, Telugu, and Bengali. This causes delays, miscommunication, and poor customer experience at the counter.

---

## 💡 Solution

VoiceAssist AI is a real-time multilingual voice assistant that:
- Automatically detects the customer's language
- Responds in the customer's native language via AI-generated voice
- Guides bank staff through complex processes step by step
- Enables agents to speak English while the customer hears their native language

## ✨ Features

- 🎙️ **Voice Input** — Customer speaks in any Indian language
- 🤖 **AI Response** — LLaMA 3.3 70B generates contextual banking replies
- 🔊 **Native Voice Output** — Sarvam AI TTS speaks in customer's language
- 🌐 **6 Languages** — Hindi, Marathi, Tamil, Telugu, Bengali, English
- 🔐 **Smart Verification** — Name + Mother's name verification for balance queries
- 🚨 **Agent Takeover** — Complex queries escalate to human agent mode
- 🎤 **Agent Voice Reply** — Agent speaks English, customer hears native language
- 📚 **RAG** — Banking FAQ knowledge base for accurate responses
- 📊 **Analytics Dashboard** — Session tracking, query types, language distribution
- 📤 **Export Sessions** — Download full conversation transcripts

## 🚀 How to Run Locally

### Backend
```bash
cd backend
pip install -r requirements.txt

# Create .env file with:
# GROQ_API_KEY=your_groq_key
# SARVAM_API_KEY=your_sarvam_key
# DEEPGRAM_API_KEY=your_deepgram_key

uvicorn main:app --reload --port 8000
```

### Frontend
```bash
cd frontend
npm install

# Create .env file with:
# REACT_APP_API_BASE=http://localhost:8000

npm start
```

## 🌊 Conversation Flow

```
Customer speaks → Whisper STT → Language Detection
       ↓
LLaMA 3.3 70B classifies query
       ↓
Simple query → AI answers directly in customer's language
Complex query → Agent Takeover Mode
       ↓
Sarvam AI TTS → Customer hears native language voice
```

**Balance Verification Flow:**
1. Customer requests balance
2. AI asks for full name
3. AI asks for mother's name
4. AI gives balance (₹45,230.50 mock)
5. Further queries handled without re-verification
